# CombatStatus

A secure-template example addon from Ch 15 and Ch 25 that performs protected actions during combat.

## Purpose
Demonstrates how to use secure templates and snippets to take protected actions (like casting spells, targeting units) that would otherwise be blocked during combat.

## Key Techniques Demonstrated
- SecureActionButtonTemplate usage
- Attribute-based action configuration
- State drivers and `RegisterStateDriver`
- Secure handler snippets (Ch 25)
- In-combat lockdown awareness
